# Términos y Condiciones - Penguin Path

**Última actualización: 13 de noviembre de 2025**

## 1. Aceptación de los términos

Al usar Penguin Path, aceptas estos términos y condiciones.

## 2. Descripción del servicio

Penguin Path es una plataforma educativa gratuita para aprender comandos de Linux a través de:
- Lecciones interactivas
- Terminal virtual aislada y segura
- Retos prácticos
- Sistema de puntuación

## 3. Uso apropiado

### Permitido:
- Aprender y practicar comandos de Linux
- Completar lecciones y retos
- Compartir tu progreso

### Prohibido:
- Intentar acceder a datos de otros usuarios
- Realizar ataques o pruebas de penetración
- Usar la plataforma para actividades ilegales
- Compartir credenciales de acceso

## 4. Terminal virtual

La terminal funciona en un entorno completamente aislado sin:
- Acceso a internet
- Acceso a otros sistemas
- Capacidad de almacenar archivos permanentemente
- Privilegios de administrador del sistema

## 5. Contenido educativo

Todo el contenido es con fines educativos. Los comandos enseñados deben usarse responsablemente.

## 6. Limitación de responsabilidad

Penguin Path se proporciona "tal cual" con fines educativos. No nos hacemos responsables por:
- Uso indebido del conocimiento adquirido
- Interrupciones del servicio
- Pérdida de progreso (aunque hacemos respaldos regulares)

## 7. Propiedad intelectual

El contenido educativo y la plataforma son propiedad de Penguin Path.

## 8. Terminación

Nos reservamos el derecho de suspender cuentas que violen estos términos.

## 9. Cambios a los términos

Notificaremos cambios importantes a través de la plataforma.
